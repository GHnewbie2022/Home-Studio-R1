## 審查結論

### 1. 框架項目確認（無需移除）

| 項目 | 來源 | 說明 |
|------|------|------|
| Blue Noise | 範例 framework | 在 PathTracingCommon.js 的 ShaderChunk 中定義，是框架標準功能 |
| Path Tracing 基本函式 | 範例 framework | QuadIntersect, BoxIntersect, BoxInteriorIntersect |
| Screen Output/Blend | 範例 framework | ScreenOutput_Fragment.glsl, ScreenCopy_Fragment.glsl |

### 2. R1 額外項目（保留）

- samplesPerFrame 控制
- CAMERA_PRESETS + switchCamera() - 多視角功能
- basicBrightness + colorTemperature + kelvinToRGB() - 燈光系統
- wallAlbedo - 牆面反射率
- acousticPanelVisibility - 聲學面板控制
- bloomIntensity + bloomRadius - Bloom 後處理
- 完整快照系統
- 自訂 GUI (setupGUI)
- 自訂鍵盤/滑鼠事件監聽

### 3. 未使用檔案

65 個 JS 範例檔案是範例 framework 本身自帶的展示程式碼，不是舊專案帶來的。以後 R2~R6 需要用到什麼功能時，可從這些範例中參考。
